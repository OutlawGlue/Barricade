## Barricade

This project is inspired by **Barricade.gg** and aims to recreate its core gameplay in a simplified, interactive format.  

The goal is to develop a game where players **strategically block opponents and control territory**, combining **tactics, timing, and planning** to win.  

The project focuses on creating a playable prototype that demonstrates the key mechanics, with scope to expand features and refine gameplay over time.

It also uses my A-star algorithm to check whether there is still a valid path for both players.
